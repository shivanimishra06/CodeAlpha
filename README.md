# 📚 Flashcard Quiz App

A simple Flashcard Quiz App built using Python and Streamlit.

## Features
- ➕ Add Flashcards
- ✏️ Edit Flashcards
- 🗑️ Delete Flashcards
- 👀 Show Answer
- ⬅️ Previous & Next Navigation
- 🎲 Random Flashcard
- 💾 Data saved using JSON

## Technologies Used
- Python
- Streamlit
- JSON

## How to Run

1. Install Streamlit:
   pip install streamlit

2. Run the app:
   streamlit run app.py
